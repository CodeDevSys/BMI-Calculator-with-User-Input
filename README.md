# BMI-Calculator-with-User-Input
Project Purpose
This project calculates the **Body Mass Index (BMI)** based on weight & height input.  
It shows if the user is **underweight**, **normal weight**, **overweight**, or **obese**.

## Technologies Used
- HTML  
- CSS  
- JavaScript  

##  How It Works
1. Enter weight (kg) and height (cm)  
2. Click the “Calculate” button  
3. The BMI value and category will be displayed

##  Example Output
Weight: 70
Height: 175
→ Your BMI is 22.86 (Normal weight)
